虽然对于初学者来说使用控制台命令的成本更高，但为了方便之后的讲解，我还是没有引入`anaconda`、`pycharm`、`Spyder`之类的工具，因为它们虽然能减少新手上手的难度，但同样会屏蔽一些过程，让初学者不方便真正理解`Python`的使用。

# 1 下载Python安装包

首先我们先访问[Welcome to Python.org-Python官方网站]([Download Python | Python.org](https://www.python.org/))，点击`Downloads`-`Windows`。

![[image-20240608085851217.png]]

下拉找到`Python3.9`且有Release文件的版本，如`3.9.13`，下载`Windows Installer(64-bit)`。

![[image-20240608090107603.png]]

需要注意的是，不同版本的`Python3`有不同的特性，一般而言是向下兼容的，我这里下载的是`3.9`版本的安装包，根据需要可以使用更新的安装包，就初学者而言，新版本的特性一般是用不上的，但不同版本的用法可能略有差异。

# 2 安装Python3

当我们下载完成后，运行安装包，开始安装。

勾选下面两个复选框，点击`Install Now`：

- 为所有用户安装：这台机器的所有用户都可以使用`Python`。
- 将`Python3.9`添加到`PATH`系统环境变量里：系统可以自动搜索到`Python`。
![[image-20240608090218111.png]]
安装完后，可能会提示`关闭最大长度限制`，关闭即可。没有提示则忽略。
![[image-20240608090458713.png]]

完成后，我们鼠标右键`开始`图标，点击`Windows Powershell`进入控制台页面。

![[image-20240608090721220.png]]

在控制台内输入

```shell
python
```

看到输出说明安装成功。

![[image-20240608090836538.png]]

此时键盘`Ctrl+Z`退出。

到这里你的运行环境就搭建完成。
